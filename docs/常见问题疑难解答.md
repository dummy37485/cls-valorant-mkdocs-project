# **常见问题疑难解答**

遇到问题先关软件游戏别退 打开DMA测速工具确保测速正常!  

- 港服挂10分钟测速确保固件未屏蔽  

测速没问题再阅读此文档!  

## **提示读取cpuid.ini失败**
阅读国服/港服开启教程  
  - 提示：每次机器码变动或做了动态机器码 需要重新采集cpuid  

## **初始化键盘失败**
软件支持Win10-Win11全系统  
若失败重启主机多试几次 仍然失败检查DMA固件是否正常  

## **港服右下角一直显示"等待VGK加密"(超过5分钟)**
原因: cpuid变了没重新采集, 或采集的是副机而非主机的cpuid信息!  
  - 仔细阅读教程文档!

## **自瞄/扳机没效果**
仔细阅读菜单Home页

## **辅机除融合器外还有显示器时，软件绘制在辅机显示器而非融合器上**
原因: 开启软件时弹出的融合器对话框没选对显示器设备
